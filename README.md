Proyecto de final de grado: Club de tiro El Parador

Este proyecto es una pagina web diseñada para la digitalizar y profesionaliazr el negocio familiar. Principalmente busca ayudar a automatizar procesos cotidianos 
y el manejo de datos de manera digital.

--Tecnologías Utilizadas--

El proyecto emplea un stack tecnológico clásico y robusto para asegurar estabilidad y compatibilidad:

    Frontend: HTML, CSS y JavaScript, para diseñar una interfaz que sea dinámica.

    Backend: PHP para realizar la lógica de servidor y el procesamiento de datos.

    Base de Datos: MySQL para gestionar la base de datos del proyecto.


--Características principales--

    Gestion de socios: registro de usuarios, incluyendo tipos de licencias y dattos personales.

    Sistema de reservas: implemenntando un calendario interactivo para reservar los diferentes puestos a tiempo real.

    Sistema de historial(Propuesta a futuro): sistema que ayude a almacenar videos a camara lenta de todas las tiradas realizadas.
